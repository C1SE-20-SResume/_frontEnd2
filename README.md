# How to use?

Run command

```bash
yarn
```

## See preview

```bash
yarn start
```

## Build production

```bash
yarn build
```

## Framework in project

- [ReactJs](https://reactjs.org/)
- [TailwindCSS](https://tailwindcss.com/)
- [React-Router](https://reacttraining.com/react-router/web/guides/quick-start)
